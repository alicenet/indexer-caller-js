Class: IdxCaller

IdxCaller(configObjectOrIndexerURL) → {null}

Index Caller is responsible for calling common end points on the AliceNet Indexer It will always return the data requested or an object with an error field eg getCurrentBlockHeight Success = 201, but a failure equals {error: "MSG"}
